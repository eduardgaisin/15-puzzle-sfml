# 15-puzzle-sfml
15 puzzle with SFML

# Installation
  ## Dependencies
    - CMake, version 3.18+
    - SFML, version 2.5+

  ## Building
  Clone repository, cd to project directory, run build script
  ```
  git clone https://github.com/eduardgaisin/15-puzzle-sfml.git
  cd 15-puzzle-sfml
  ./build.sh
  ```

  ## Launching
  To run game, navigate to `build/target` folder and run `15-puzzle-sfml`
  ```
  cd build/target/
  ./15-puzzle-sfml
  ```

# Controls

R - shuffle board
