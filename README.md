# 15-puzzle-sfml
15 puzzle with SFML

# Installation
    ### Building project
    '''
    git clone https://github.com/eduardgaisin/15-puzzle-sfml.git
    cd 15-puzzle-sfml
    ./build.sh
    '''

    ### Launching
    '''
    cd build/target/
    ./15-puzzle-sfml
    '''

# Controls

R - shuffle board
